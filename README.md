# Customer---Support---Analytics---Dashboard
An interactive Power BI dashboard for analyzing customer support performance, ticket trends, customer satisfaction, ticket priorities, and support channels using KPI cards, gauge charts, and advanced data visualizations.

## Key Features
- Customer Satisfaction Gauge Chart
- Ticket Trend Analysis
- Ticket Priority Distribution
- Ticket Channel Analysis
- Ticket Status Analysis
- Interactive Filters & Slicers
- KPI Cards for Business Insights

## Tools Used
- Power BI
- Power Query
- DAX
- Data Visualization

## Dashboard Insights
- Track customer satisfaction ratings
- Analyze support ticket trends
- Monitor ticket resolution performance
- Identify high-priority support issues
- Compare ticket channels and statuses
